Privacy Policy

This Privacy Policy explains how sheetsservice collects, uses, and protects user information when accessing the YouTube Data API v3.

Information Accessed

sheetsservice may access the following information only after explicit user consent:

  • YouTube account basic information
  • YouTube channel details
  • Video and playlist metadata

sheetsservice does not collect Google account passwords or any unrelated personal information.

Use of Information

All accessed information is used solely to provide the intended functionality of the application, including:

  • Displaying YouTube channel data
  • Managing authorized YouTube resources
  • Ensuring proper operation of API features

Data Storage & Security

OAuth access tokens and refresh tokens are stored securely and used only for authorized API requests. sheetsservice does not permanently store unnecessary user data.

Data Sharing

sheetsservice does not sell, rent, or share user data with third parties. All data usage strictly follows the Google API Services User Data Policy.

User Control

Users can revoke sheetsservice’s access at any time via their Google Account security settings. Once access is revoked, the application will no longer be able to access YouTube data.

Policy Updates

This Privacy Policy may be updated periodically. Any changes will be posted on this page.
