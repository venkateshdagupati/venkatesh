sheetsservice

sheetsservice is a web-based application that integrates with the YouTube Data API v3 to allow users to securely access and manage their YouTube account data after explicit authorization.

Application Purpose

The purpose of sheetsservice is to provide a simple and reliable way for users to retrieve, view, and manage YouTube-related data such as channel information, video metadata, and other authorized resources. The application is intended for personal use, testing, automation, and data management purposes.

YouTube API Usage

sheetsservice uses the YouTube Data API v3 to:

  • Authenticate users via Google OAuth 2.0
  • Access YouTube channel and account information
  • Read and manage authorized YouTube data
  • Perform API requests strictly within granted permissions

User Data & Authorization

All access to YouTube data is performed only after the user grants permission through Google’s official OAuth consent screen. sheetsservice only requests the minimum scopes required for its functionality.

Data Security

sheetsservice takes user privacy seriously. OAuth access tokens and refresh tokens are stored securely and are used only to perform authorized YouTube API requests. No user data is sold, shared, or misused.

Hosting & Transparency

This application is hosted on GitHub Pages to ensure transparency and accessibility. The source code and documentation are maintained in the GitHub repository.

Contact

For questions, concerns, or support regarding sheetsservice, please contact the developer through the GitHub repository associated with this application.
